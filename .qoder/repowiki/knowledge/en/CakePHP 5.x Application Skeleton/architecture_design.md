This is the top-level CakePHP application skeleton (package `cakephp/app`). The single HTTP entry `index.php` delegates to `webroot/index.php`, which boots the framework; all application code lives under `src/` (PSR-4 namespace `App\`) and tests under `tests/` (`App\Test\`). Runtime dependencies are declared in `composer.json`: CakePHP 5.3 core with the authentication and authorization plugins, migrations, and MobileDetect. Dev-only tooling is wired through Composer scripts — `@check` runs both PHPUnit and PHPCS, `@cs-fix` invokes PHPCBF, and a `post-install-cmd` / `post-create-project-cmd` hook calls `App\Console\Installer::postInstall`. Static analysis and quality gates are configured separately via `phpstan.neon` (level 8), `psalm.xml`, `phpunit.xml.dist`, and `phpcs.xml` (using `cakephp/cakephp-codesniffer`). Environment overrides go into `config/app_local.php` while base config is in `config/app.php`. Deployment is guided by `.htaccess` (Apache) and `.gitignore` excludes vendor/config/runtime artifacts.
